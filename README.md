# Zoo Veri Seti ile Karar Ağacı Sınıflandırması

Bu projede UCI Machine Learning Repository'de bulunan Zoo Veri Seti kullanılarak Decision Tree (Karar Ağacı) algoritması ile hayvan sınıflandırması yapılmıştır.

## Veri Seti

- Veri Seti: Zoo Dataset
- Toplam Örnek Sayısı: 101
- Özellik Sayısı: 16
- Sınıf Sayısı: 7

## Kullanılan Teknolojiler

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Kullanılan Değerlendirme Ölçütleri

- Accuracy (Doğruluk)
- Precision (Kesinlik)
- Recall (Duyarlılık)
- F1-Score
- Confusion Matrix

## Görselleştirmeler

- Karar Ağacı Görselleştirmesi
- Feature Importance Grafiği
- Confusion Matrix
- Max Depth Analizi

## Sonuçlar

Decision Tree algoritması ile yaklaşık %95 doğruluk oranı elde edilmiştir.
